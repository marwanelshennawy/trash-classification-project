# trash-classification-project
A deep learning project for classifying trash into 10 categories 
----
This project presents a deep learning-based image classification system designed to automatically categorize trash into 10 different waste types. The goal of this project is to support smart waste management and recycling systems by helping machines identify different types of garbage from images.

The model is trained on a dataset containing labeled images of various waste categories. A Convolutional Neural Network (CNN) is used to learn visual patterns and features that distinguish different types of trash.

By accurately classifying waste materials, this system can contribute to automated recycling, environmental protection, and efficient waste sorting.

🚀 Features

Classifies trash images into 10 categories :
├── battery/      (755 images)
├── biological/   (797 images)
├── cardboard/    (1,460 images)
├── clothes/      (4,261 images)
├── glass/        (2,645 images)
├── metal/        (816 images)
├── paper/        (1,525 images)
├── plastic/      (1,587 images)
├── shoes/        (1,818 images)
└── trash/        (757 images)


Built using Deep Learning and Computer Vision techniques

Uses Convolutional Neural Networks (CNNs) for feature extraction and classification

Supports training, validation, and prediction

Designed to assist smart recycling systems
